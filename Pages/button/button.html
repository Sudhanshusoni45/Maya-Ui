Button

Buttons are one of the most used and most important component, they allow users to take actions and make choices with a single tap.

Contained Buttons

Contained Buttons are high emphasis buttons.

Outlined Buttons

Outlined buttons are lower emphasis button then contained buttons.

Link Button

Link buttons are used to attach a actionable link at button tap.

Icon Button

We recognize pictures more easily and faster then text so for that reason icon buttons are a good choice.

Floating Action Button

A Floating Action Button is a circular button used to perform the primary action of the screen it is present in.
