Input

Avatar are circular images generally used to represent a user or user's persona.

Label Input

Profile avatars can created by adding class = 'avatar' inside img tag.

Success Input

Avatars comes in three different sizes: small, medium and large. To use these sizes just add their abbreviation sm, md, lg respectively along with 'avatar' class.
Note: Default avatar size is medium.
